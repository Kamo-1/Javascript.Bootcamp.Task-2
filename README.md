# Student Grade Check

A lightweight, single-file web app for checking and recording student grades.

---

## Features

- Enter a student's name and mark (0–100) to instantly generate a grade result
- Grades are classified as **Distinction**, **Merit**, **Pass**, or **Fail**
- Each submitted student is added to a running list with their mark and grade badge
- **Clear All** button to wipe the list

---

## Grade Boundaries

| Mark Range | Grade       | Status |
|------------|-------------|--------|
| 80 – 100   | Distinction | Pass   |
| 65 – 79    | Merit       | Pass   |
| 50 – 64    | Pass        | Pass   |
| 0 – 49     | Fail        | Fail   |

---

## Usage

1. Open `student-grade-check.html` in any modern browser
2. Enter the student's name and mark, then click **Submit **
3. The result appears immediately below the form
4. The student is added to the list
5. Use **Clear list** to reset everything

---

---

## Requirements
